### Selected Projects

**6G Threat Detection** — MSc dissertation. Dual-layer firewall with ML-based threat classification, NS-3 (C++) smart-city network simulation, and a React dashboard with a human-in-the-loop operator review workflow. Operator decisions feed back as labelled training data so the model improves over time. [Source](https://github.com/itsmesaim/AI-firewall)

**FastAPI Backend Suite** — Three Python REST APIs (F1 comparison, financial management with a Kanban board, task management) with JWT auth, structured routing, and auto-generated OpenAPI docs.

**PONG-MERN** — Real-time multiplayer game. React, Node.js, and Socket.IO with authoritative server-side physics, room-based matchmaking, reconnect handling, and mobile touch controls.
